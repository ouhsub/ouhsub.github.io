内置函数

  1. Misc Function(大杂烩)

    • color

      将颜色字符串转化为颜色值
      参数: 特定颜色字符串
      返回值: 颜色值
      例子: color("#aaa");
      输出: #aaa

    • image-size

      从文件获取图像尺寸
      参数: 需要获取尺寸的文件(字符串)
      返回值: 尺寸
      例子: image-size("file.png");
      输出: 10px 10px
      注意: 需要node执行环境。
      添加版本: v2.2.0

    • image-width

      获取图像宽度
      参数: 需要获取尺寸的文件(字符串)
      返回值: 宽度值
      例子: image-width("file.png");
      输出: 10px
      注意: 需要node执行环境
      添加版本: v2.2.0

    • image-height

      获取图像高度
      参数: 需要获取尺寸的文件(字符串)
      返回值: 高度值
      例子: image-height("file.png");
      输出: 10px
      注意: 需要node执行环境
      添加版本: v2.2.0

    • convert

      转换数值单位
      参数: 第一个参数是一个包含单位的数值，第二个参数是单位。 如果单位是兼容的，数值将会被转换。如果单位不兼容，那就原样返回。
      兼容的单位组: 

      长度: m, cm, mm, in, pt and pc
时间: s and ms
角度: rad, deg, grad and turn
      返回值: 数值
      例子: 
      convert(9s, "ms") //=>9000ms
convert(14cm, mm) //=>140mm
convert(8, mm) //=>8 incompatible unit types

    • data-uri

      Inlines a resource and falls back to url() if the ieCompat option is on and the resource is too large, or if you use the function in the browser. If the MIME type is not given then node uses the mime package to determine the correct mime type.
      参数: mimetype: (Optional) A MIME type string. url: The URL of the file to inline.
      If there is no mimetype, data-uri function guesses it from filename suffix. Text and svg files are encoded as utf-8 and anything else is encoded as base64.
      If user provided mimetype, the function uses base64 if mimetype argument ends with ;base64. For example, image/jpeg;base64 is encoded into base64 while text/html is encoded into utf-8. 

      //例子:
data-uri('../data/image.jpg'); //=>url('data:image/jpeg;base64,bm90IGFjdHVhbGx5IGEganBlZyBmaWxlCg==');//Output in browser: url('../data/image.jpg');
data-uri('image/jpeg;base64', '../data/image.jpg'); //=>url('data:image/jpeg;base64,bm90IGFjdHVhbGx5IGEganBlZyBmaWxlCg==');
data-uri('image/svg+xml;charset=UTF-8', 'image.svg'); //=>url("data:image/svg+xml;charset=UTF-8,%3Csvg%3E%3Ccircle%20r%3D%229%22%2F%3E%3C%2Fsvg%3E");
