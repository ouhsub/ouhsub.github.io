概述

Less作为CSS的预编译语言，一切特性都是为了构建逻辑性强，结构良好，复用、扩展、维护方便的样式表。

less的一些基本概念

  1. 变量

    变量的定义方式是: @变量名: 变量值;

  2. 混入

    Mixin本质上就是复制一份规则列表到目标选择器上，是实现代码复用的一种方式。

  3. 嵌套

    嵌套是对样式表进行结构化组织的一种方式。Less中既可以对选择器进行嵌套， 也可以对指令(@media,@supports，keyframe等)进行嵌套。

  4. 运算

    Less中可以对数值、颜色、变量等进行+、-、*、/等运算。

  5. Escaping

    使用～符号可以使用任意字符串作为属性值或者变量值，除了变量插值外，保持字符串不变。

  6. 内置函数

    Less支持大量内置函数用来处理颜色、字符串等。

  7. 命名空间和访问器

    为了更好组织 CSS 或者更好的封装，将一些变量或者混合模块打包起来，以重复使用

  8. 作用域

    实际山就是函数的作用域。

  9. 注释

    Less支持两种注释方式: 

    //行内注释，不会被编译输出
/* 注释块，会被编译到CSS文件中 */

  10. 导入文件

    组织代码结构的利器。
